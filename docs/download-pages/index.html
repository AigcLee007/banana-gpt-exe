下载前建议看一下

Windows 怎么装 下载 `.exe` 后双击安装。如果系统提示安全确认，选择继续安装即可。
Mac 怎么装 下载 `.dmg` 后打开安装。若首次启动被系统拦截，可到“隐私与安全性”中允许打开。
Mac 怎么看芯片 点击左上角苹果菜单，选择“关于本机”。看到 M1/M2/M3/M4 就选 Apple 芯片版。
